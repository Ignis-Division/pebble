Privacy Policy

Effective date: February 25, 2026  ·  Last updated: February 25, 2026
TL;DR

ModFilter reads member data directly from Discord to perform moderation actions. It does not store, sell, or share your personal data. No data is retained after a moderation action is complete.

1. Who We Are

ModFilter ("the Bot") is a Discord moderation bot developed and operated by an independent developer ("the developer"). This Privacy Policy explains what data the Bot accesses, why, and how it is handled.

2. Data We Access

The Bot accesses the following data via the Discord API in order to function. This data is accessed in real-time and is not stored persistently.

Data Why it's accessed Stored?
Discord User ID To identify members for moderation actions No
Display name / username To display member information in filter results No
Profile avatar (presence or absence) To filter members without a profile photo No
Server roles To filter members by role No
Server join date To filter members by how recently they joined No
Guild (server) ID To maintain the whitelist of approved servers Yes — whitelist only

The only data stored on disk is a list of approved server (guild) IDs and their access expiry dates. No personal user data is ever written to disk.

3. How We Use Data

Accessed data is used solely for the following purposes:

  • Displaying filtered member lists to server moderators
  • Executing moderation actions (kick, ban, mute, role removal, DM warning)
  • Logging moderation events to a designated developer log channel

Data is never used for advertising, analytics, profiling, or any commercial purpose.

4. Data Sharing

We do not sell, rent, or share any data with third parties. Member data accessed through the Discord API is not transmitted to any external service.

The Bot operates entirely within the Discord platform using Discord's own API. All data handling is subject to Discord's Privacy Policy in addition to this policy.

5. Data Retention

Member data (usernames, IDs, roles, avatars, join dates) is accessed in real-time from Discord and is never written to any database or file. It exists only transiently in memory during the execution of a command and is discarded immediately after.

The only persistent data stored is:

  • Approved server IDs and their whitelist expiry dates (stored in a local JSON file)

This data contains no personal user information and is deleted automatically when a server's whitelist expires or when the server is removed.

6. Security

The Bot is hosted on a private server accessible only to the developer. No member data is exposed via any public-facing API or web interface. The whitelist file containing server IDs is stored locally and not transmitted externally.

7. Children's Privacy

The Bot is not directed at children under the age of 13. We do not knowingly collect any information from children. If you believe a child under 13 has used this Bot, please contact us so we can take appropriate action.

8. Your Rights

Since no personal data is stored by the Bot, there is no personal data to access, correct, or delete. If you have concerns about data processed by Discord itself, please refer to Discord's Privacy Policy.

To request removal of your server from the whitelist or to ask any privacy-related questions, contact the developer via Discord.

9. Changes to This Policy

This Privacy Policy may be updated from time to time. The effective date at the top of this page will be updated accordingly. Continued use of the Bot after changes are posted constitutes acceptance of the revised policy.

10. Contact

For privacy questions or concerns, contact the developer via Discord or through the contact information listed on the Bot's Discord Developer Portal page.
